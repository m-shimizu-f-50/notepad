# Ruby ハッシュ(連想配)

## ハッシュとは？
ハッシュとは、まとまったデータを便利に扱うことができるデータ構造のひとつ。
キーと呼ばれる文字列で、それぞれのデータを区別します。

## ハッシュの基本

```
配列のおさらい

enemy_array=["スライム","モンスター","ドラゴン"]
p enemy_array   //出力結果　["スライム","モンスター","ドラゴン"]
puts enemyarray[0]    //スライム



ハッシュの具体例
enemy_hash={"ザコ"　=> "スライム", "普通" =>"戦士", "ラスボス" => "魔法使い"}
p enemy_hash    // {"ザコ"　=> "スライム", "普通" =>"戦士", "ラスボス" => "魔法使い"}
puts enemy_hash["普通"]      //戦士
puts enemy_hash.length      //3(配列の中の要素の数を数えている)

level = "ザコ"
puts enemy_hash[level]      //ザコ

＊ハッシュは波括弧{}で囲みカンマ(,)で区切ります



ハッシュタグの追加
enemy_hash["ザコ2"]="メタルモンスター"
p enemy_hash    // {"ザコ"　=> "スライム", "普通" =>"戦士", "ラスボス" => "魔法使い"　,"ザコ2" => "メタルモンスター"}


ハッシュタグの編集
enemy_hash["普通"] = "レッドドラゴン"
p enemy_hash    // {"ザコ"　=> "スライム", "普通" =>"レッドドラゴン", "ラスボス" => "魔法使い"　,"ザコ2" => "メタルモンスター"}


ハッシュタグの削除
enemy_hash.delete("ザコ")
p enemy_hash    // {"普通" =>"レッドドラゴン", "ラスボス" => "魔法使い"　,"ザコ2" => "メタルモンスター"}
```

## ハッシュの整列
sortメソッドを使って並び方を変える


```
items = {"イージスシールド" =>40,"ウィンドスピア"=>12,"イージスシールド" =>40}

p items
//{"イージスシールド" =>40,"ウィンドスピア"=>12,"アースブレイカー"=>99}


p items.sort
//[["アースブレイカー",99],["イージスシールド",40],["ウィンドスピア",12]]  配列であいうえお順で呼び出している


p items.sort.to_h
//{"アースブレイカー"=>99,"イージスシールド" =>40,"ウィンドスピア"=>12}
ハッシュで値の大きい順に呼び出されている


p items.sort_by {|k,v| v}
//[[["ウィンドスピア",12],["イージスシールド",40],["アースブレイカー",99]]



p items.sort_by {|k,v| v}.to_h
//{"ウィンドスピア"=>12,"イージスシールド" =>40,"アースブレイカー"=>99}

```




## ハッシュにシンボルを使う

```
シンボルを使わない
enemise1 = {"ザコ" => "スライム","中ボス"　=> "ドラゴン", "ラスボス" => "魔王"}
p enemise1
p enemise1["ザコ"]

//出力結果
{"ザコ" => "スライム","中ボス"　=> "ドラゴン", "ラスボス" => "魔王"}
スライム




//シンボルを使用
enemise1 = {:zako => "スライム",:boss　=> "ドラゴン", :king => "魔王"}
p enemise1
p enemise1[:boss]

//出力結果
{:zako => "スライム",:boss　=> "ドラゴン", :king => "魔王"}
ドラゴン


enemise1 = {zako: "スライム",boss:"ドラゴン", king:"魔王"}
p enemise1
p enemise1[:boss]


//出力結果
{zako: "スライム",boss:"ドラゴン", king:"魔王"}
ドラゴン
```

＊記述がシンプルになる
